MeBooks Registry Integration Test

This page tests the integration between a catalog registry viewer and MeBooks. Click the test buttons below to add OPDS catalogs to an existing MeBooks instance or open a new one.

Expected behavior: If MeBooks is already open, it should automatically add the catalog. Otherwise, it will open MeBooks and add the catalog.

Status Check

How to Test:

  1. First, open MeBooks in another tab: Open MeBooks
  2. Click one of the test buttons above
  3. If MeBooks is already open, it should receive the catalog and show a toast message
  4. If MeBooks isn't open, it will open automatically with the catalog
  5. The catalog should appear in your library's catalog list
