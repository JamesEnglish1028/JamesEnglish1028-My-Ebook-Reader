This page tests the integration between a catalog registry viewer and MeBooks. Click the test links below to launch MeBooks with different OPDS catalogs.

How to Test:

  1. Click one of the test links above
  2. MeBooks should open in a new tab/window
  3. You should see a toast message saying "Successfully added catalog: [Name]"
  4. The catalog should appear in your library's catalog list
  5. The URL parameters should be cleaned from the address bar
